Study Up Coding

Study Up Coding is an application built to help users save tutorials they have found online and share them. Includes authentication using passport. Built with Angular, Node and Express. Also uses Skeleton as a front-end framework in place of Bootstrap. I made use of nested schemas in mongo to help develop a relational database. I used a lot of tests to make sure all my routes on the back end were working properly with my database before starting on the front end.
